# 14. Availability Calendar

You are given a simple list of days standing in for a calendar grid.

Implement:

1. Range selection with a start and end date.
2. Prevention of selecting booked days.
3. A clear visual indication of the selected range.
4. A summary of the chosen availability.

Notes:

- You may keep the list UI or convert it into a more calendar-like layout.
- Be ready to explain how you would model range state and edge cases.
