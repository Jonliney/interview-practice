# 08. File Explorer

You are given a tiny file explorer for nested folders and files.

Implement:

1. Expand and collapse folders.
2. A text search that only shows matching branches.
3. Renaming a file or folder.
4. Immutable nested updates.

Notes:

- The current starter keeps tree data in a single nested structure.
- The rename path is a good place to discuss recursive helpers and type-safe tree operations.
